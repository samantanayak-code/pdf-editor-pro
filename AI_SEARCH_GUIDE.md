# AI Search & Citations - User Guide

## Overview

The AI Search feature allows you to search across all your uploaded PDF documents and get accurate citations with page numbers. This is especially useful for contract managers, researchers, and anyone who needs to find specific information across multiple documents.

## Getting Started

### 1. Upload Your PDFs

1. Navigate to the **AI Search** tab in the main menu
2. Click the **Upload PDF** button (top right)
3. Select a PDF file from your computer
4. Wait for the upload and indexing to complete (this may take a few seconds depending on file size)
5. Your document will appear in the "Your Documents" section

**What happens during upload:**
- The PDF is uploaded to secure cloud storage
- Text is extracted from every page
- Content is indexed for fast searching
- Page references are preserved for citations

### 2. Search Your Documents

#### Basic Search
1. Type your search query in the search box
2. Press Enter or click the "Search" button
3. Results appear below with highlighted matches

#### Search Specific Documents
1. Use the dropdown menu to select a specific document
2. Or leave it on "All documents" to search everywhere
3. Enter your search query and press Enter

### 3. Understanding Search Results

Each search result shows:

```
┌─────────────────────────────────────────┐
│ 📄 Contract_ABC.pdf         Page 5      │
│                          85% match      │
├─────────────────────────────────────────┤
│ Your search terms will be highlighted   │
│ in yellow within the matching text      │
│ context from that specific page.        │
├─────────────────────────────────────────┤
│ Citation: Contract_ABC.pdf, Page 5      │
└─────────────────────────────────────────┘
```

**Result Components:**
- **Document Name**: The source PDF filename
- **Page Number**: Exact page where the text was found
- **Match Percentage**: How well the content matches your query
- **Highlighted Text**: Your search terms highlighted in context
- **Citation**: Ready-to-copy citation format

## Search Tips

### Effective Searching

**1. Use Specific Keywords**
```
Good: "force majeure clause"
Less effective: "clause"
```

**2. Search Multiple Terms**
```
Good: "payment terms 30 days"
Finds: Pages containing all these words
```

**3. Search Phrases**
```
Good: "governing law jurisdiction"
Finds: Pages with these related terms
```

**4. Use Context Words**
```
Good: "contractor obligations safety"
Better than: "obligations"
```

## Troubleshooting

### No Results Found
**Problem**: Search returns no results

**Solutions:**
1. Check spelling of search terms
2. Try synonyms or related terms
3. Search with fewer keywords
4. Verify the document is uploaded

### Upload Failed
**Problem**: PDF won't upload

**Solutions:**
1. Check file size (max 100MB)
2. Ensure file is a valid PDF
3. Try a different browser
4. Check internet connection

### Text Not Extracted
**Problem**: Document uploaded but no search results

**Possible Causes:**
1. Scanned PDFs (images of text) - OCR coming soon
2. Encrypted or password-protected PDFs
3. Corrupted PDF files

**Solution**: Try re-saving the PDF or using a different source
