# PDF Editor Pro - Quick Start Guide

## Getting Started in 3 Minutes

### Step 1: Sign In (10 seconds)
1. Open the application
2. Click "Sign In" or "Get Started Free"
3. Create account with email/password
4. You're in!

### Step 2: Open a PDF (30 seconds)
1. Click the **"Pro Editor"** tab (with sparkles icon ✨)
2. Click **"Open PDF"** button
3. Select your PDF file
4. Wait for loading (usually 1-3 seconds)
5. Your PDF appears on the canvas!

### Step 3: Edit Your Document (2 minutes)
1. **Select Mode** is active by default
2. Hover over text - you'll see blue highlights
3. Click on any text element to select it
4. Use the toolbar to:
   - Change text properties
   - Move elements
   - Delete elements
5. Use `Ctrl+Z` to undo mistakes

### Step 4: Export (10 seconds)
1. Click **"Export"** button (green, top right)
2. Wait for processing (1-3 seconds)
3. Your edited PDF downloads automatically!
4. Done!

## Toolbar Quick Reference

### Top Toolbar
- **Open PDF**: Load a PDF document
- **Export**: Download edited PDF

### Edit Toolbar
- **🖱️ Select (V)**: Click to select text elements
- **✋ Pan (H)**: Drag to move around the page
- **📝 Text (T)**: Add new text (coming soon)

### History Toolbar
- **↶ Undo (Ctrl+Z)**: Undo last change
- **↷ Redo (Ctrl+Y)**: Redo change

### Format Toolbar (when text selected)
- **B**: Bold text
- **I**: Italic text
- **U**: Underline text

### Zoom Toolbar
- **🔍-**: Zoom out (Ctrl+-)
- **100%**: Current zoom level
- **🔎+**: Zoom in (Ctrl++)

## Keyboard Shortcuts Cheat Sheet

### Editing
- `V` - Switch to Select mode
- `H` - Switch to Pan mode
- `T` - Switch to Text mode (planned)

### Navigation
- `Page Up` / `Page Down` - Navigate pages
- `Arrow Keys` - Move selected element
- `Delete` - Delete selected element

### History
- `Ctrl+Z` / `Cmd+Z` - Undo
- `Ctrl+Shift+Z` / `Cmd+Shift+Z` - Redo
- `Ctrl+Y` / `Cmd+Y` - Redo (alternative)

### View
- `Ctrl++` / `Cmd++` - Zoom in
- `Ctrl+-` / `Cmd+-` - Zoom out
- `Ctrl+0` / `Cmd+0` - Reset zoom (planned)

### File
- `Ctrl+O` / `Cmd+O` - Open PDF (planned)
- `Ctrl+S` / `Cmd+S` - Export PDF

```

```

## Tips & Tricks

### Speed Tips
- 💡 Use keyboard shortcuts instead of mouse clicks
- 💡 Press `H` to quickly pan around large documents
- 💡 Double-click to zoom to fit width (planned)
- 💡 Use `Ctrl+Z` freely - unlimited undo!

### Quality Tips
- 💡 Work at 100% zoom for precision
- 💡 Check alignment with grid (planned)
- 💡 Preview before exporting
- 💡 Keep backups of original PDFs

### Performance Tips
- 💡 Close other browser tabs
- 💡 Work on one page at a time
- 💡 Use smaller file sizes when possible
- 💡 Export regularly to save progress

## What PDFs Work Best?

### ✅ Excellent Support
- Text-based PDFs
- Single-column documents
- Standard fonts (Arial, Times, Helvetica)
- Documents under 50 pages
- File size under 20MB

### ⚠️ Good Support
- Multi-column layouts
- Custom fonts (with fallback)
- Documents 50-100 pages
- File size 20-50MB
- Simple tables

### ❌ Limited Support
- Scanned documents (no text layer)
- Complex forms
- Password-protected PDFs
- Very large files (>100MB)
- Documents with extensive annotations

## Troubleshooting Quick Fixes

```
✓ Check text layer exists (not scanned PDF)
```

```
✓ Wait a few seconds

✓ Try simpler PDF first
✓ Clear browser cache
```

### Slow Performance
```
✓ Close other tabs
✓ Reduce zoom level
✓ Work with smaller files
✓ Restart browser
```

## FAQ

**Q: Is my PDF data secure?**
A: Yes! All processing happens in your browser. No data is uploaded to servers.

**Q: What file size limit?**
A: 100MB recommended, larger files may be slow.

**Q: Can I edit scanned PDFs?**
A: Not yet. OCR feature is planned for future release.

**Q: Are my edits reversible?**
A: Yes! Use Ctrl+Z to undo. Original PDF is never modified.

**Q: Can I edit on mobile?**
A: Desktop browsers work best. Mobile support is limited.

**Q: How do I report a bug?**
A: Use GitHub Issues or contact support.
